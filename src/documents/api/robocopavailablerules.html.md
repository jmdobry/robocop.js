## Description
Return the list of rules registered with robocop.js.

## Usage
```javascript
// rules provided by robocop
robocop.availableRules();   //  [
							//      "nullable",
							//      "max",
							//      "min",
							//      "maxLength",
							//      "minLength",
							//      "type"
							//  ]
```
