## Description
Return the list of data types registered with robocop.js.

## Usage
```javascript
// data types provided by robocop
robocop.availableDataTypes();   //  [
								//      "string",
								//      "number",
								//      "integer",
								//      "float",
								//      "array",
								//      "object",
								//      "boolean",
								//      "date"
								//  ]
```
